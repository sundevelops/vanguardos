## 2. Concrete improvements (4)

1. **Audience-aware avatar selector (felt personalization, zero server).** The "Who this is for" grid was a passive label list. Each avatar is now a real control: tap "Side hustler turning serious" and the panel below resolves to copy written for *that* path — naming the specific objection (duct-taped pages, no real offer) and the specific outcome — plus a path-scoped CTA. This converts a generic "is this for me?" glance into a one-tap personalized pitch, which is the highest-leverage on-page lift available before testimonials exist. Implemented with `{#key selectedAvatar}` so the panel re-mounts and replays its entrance animation on each switch; `aria-live="polite"` + `aria-pressed` keep it accessible; clicking the active avatar again toggles back to neutral.
2. **Honest founding-buyer placeholder (trust without faking).** Added a quiet, dashed-border `#testimonials` section that openly states the product is new and refuses to fabricate reviews, then reframes the empty wall as an invitation ("be the first quote"). The skeptical early buyer is the one scanning for fake five-stars — naming the absence directly is more credible than hiding it, and the CTA turns the gap into a reason to buy now. Replaceable with real quotes the moment they exist; no schema/markup debt.
3. **Graceful day-card degradation (aesthetic credibility).** The 5-day journey is the page's main proof block, but `day-1.png`…`day-5.png` don't exist yet (only legacy webp thumbnails are in `static/`), so every card was rendering a near-empty dark square — reading as "broken" to a first-time visitor. Each card now shows a designed branded tile (big gold numeral `01`–`05` + kicker) until the real mockup lands, at which point the image simply covers it. The section now looks intentional whether or not the visual-generation task has run. Coordinates correctly with that parallel task — no code change needed when the PNGs arrive.
4. **Hero pill rotation + card hover system.** Rotated the hero eyebrow to `IDEA TO LIVE IN 5 DAYS` (outcome-forward, per the SKILL rotation list — see §6 for rotation state). Separately, the new avatar and day cards arrived flat; added a cohesive hover language (gold border bloom, subtle lift/scale, soft gold shadow) matching the existing offer-stack card feel, with a keyboard `focus-visible` ring on the now-interactive avatars and a full `prefers-reduced-motion` opt-out.
