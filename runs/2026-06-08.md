## 2. Concrete improvements (3)

1. **Specimen click-to-expand modal (queued recommendation #1 from 06-07).** The `#specimens` section previously showed palettes and type pairings but you couldn't inspect them. Each is now a keyboard-accessible button: clicking a palette opens a modal with a full-width swatch band plus a tile per color showing its **hex value**; clicking a type pairing renders the face at **display size** with the body-font description. This turns "here's a taste" into "here's the actual substance" — the strongest proof-of-quality move available before testimonials/mockups exist, and it deepens engagement (a click + dwell) on the exact section that answers the "is there really anything in here?" objection. Built with pure Svelte state, focus save/restore, Escape-to-close, and overlay-click dismiss — reusing the existing modal conventions so it reads as native.

2. **Persistent mobile CTA bar (new conversion lever).** On mobile the only buy button scrolls away with the hero/nav, leaving long stretches of the page with no tap-target at the moment of peak intent. A fixed bottom bar now slides up once the hero is ~85% scrolled past, showing the **$129 / launch-week $103 · VOS20** anchor and a `Build My System →` button straight to Gumroad. It's `md:hidden` (mobile-only, so desktop is untouched), respects the iOS safe-area inset, is removed from the tab order while hidden (`tabindex`/`aria-hidden` gated on visibility), and its slide transition is disabled under `prefers-reduced-motion`. Persistent mobile CTAs are one of the most reliable conversion lifts on long sales pages — this adds one without disturbing the desktop layout.

3. **Hero pill rotation (future-goal "Test hero pill copy").** Rotated the eyebrow from `IDEA TO LIVE IN 5 DAYS` → **`★ NEW · THE OPERATOR'S LAUNCHPAD`**. This run rotated the *pill* (the 06-07 run rotated the *CTA*), keeping one headline lever per pass so the moving variable stays isolated. "The Operator's Launchpad" leans into the audience's self-image — the busy professional / side-hustler who sees themselves as an operator, not a "course student" — reinforcing the brand's systems-not-content positioning right above the headline.
